# Perfect Paws Dog Grooming, Penzance — demo site

Demo marketing website for **Perfect Paws Dog Grooming Penzance** (groomer Kate, Heamoor, Penzance).

Static site — plain HTML & CSS, no build step. Open `index.html` in a browser to preview.

## Structure

| Path | Purpose |
|------|---------|
| `index.html` | The page — content, layout, contact details |
| `styles.css` | Brand styling (forest green + terracotta) |
| `images/` | Photos and logo |
| `README.txt` | Plain-English guide (adding photos, sending the demo) |

## Status

This is a **demo/pitch** build. Gallery and About photos are placeholders, and two of the three
reviews are flagged as placeholders (Helen Foster's is a real Facebook review). Add Perfect Paws'
own photos and real reviews before final handover.

## Hosting

Any static host works — drag the folder onto [Netlify Drop](https://app.netlify.com/drop) for an
instant shareable link, or connect this repo to Netlify for auto-deploys.
